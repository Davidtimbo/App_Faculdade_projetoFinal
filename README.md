# 📱 Daily Tasks App

Um aplicativo Android desenvolvido em **Java** como projeto de faculdade, que permite gerenciar tarefas do dia a dia de forma simples e eficiente.  
Com ele, é possível **criar, visualizar, editar e excluir tarefas** — um CRUD completo utilizando **SQLite** para persistência dos dados.

---

## 🚀 Funcionalidades

- ✅ **Adicionar nova tarefa** com:
  - Título  
  - Descrição  
  - Prioridade (1 a 5)  
  - Data de vencimento  

- ✏️ **Editar tarefa existente** diretamente da lista  
- 🗑 **Excluir tarefas concluídas**  
- 💾 **Banco de dados local (SQLite)** para salvar as informações  
- 📋 **Lista de tarefas** exibida na tela inicial  
- 🔄 **Atualização automática** da lista ao salvar/editar  

---

## 🛠 Tecnologias utilizadas

- **Java (Android)**
- **SQLite** para persistência de dados  
- **DAO Pattern** para acesso ao banco  
- **Adapter + ListView** para exibição dinâmica
